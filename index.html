Giving a voice

Cboard is a web application for children and adults with speech and language impairment, aiding communication with pictures and text-to-speech.

Start Cboard
Learn more ▼

Built for the web

Cboard works on modern browsers and is available on a wide variety of platforms, including desktops, tablets and mobile phones. Offline support is available on Google Chrome (desktop & android).

Personalized boards

With more than 3400 symbols from the Mulberry Symbol Set, you can create your own custom boards for different situations in life.

33 languages

Cboard comes with support for 33 languages, support varies between operating systems.

Join us on GitHub

Cboard is an open-source project and we always appreciate a helping hand, if you want to contribute you are invited to our GitHub repo.
